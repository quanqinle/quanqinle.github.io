Shell | 删除最老的一份备份

备份文件夹多于5个的话，删除最老的那个文件夹

假设：/home/backup/下会定时生成备份目录，为省空间，最多允许保存5份备份，如果超过则删除最老的一份。

讲解
grep ‘^d’: 筛选出文件夹（因为文件夹属性信息是d开头）
wc -l : 统计行数，即，统计有多少个文件夹
-ge : 大于等于
ls -tr : 按从旧到新排序，旧的排在前面
head -n1 : 选取排列的第一个文件，1可以按需改成其他数值
